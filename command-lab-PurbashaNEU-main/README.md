### Overview 
Used the Command Design Pattern to implement a batch calculator for computations as follows: 
 * adding 6 and 3 
 * subtracting 3 from 6 
 * multiplying 6 and 3 
 * dividing 6 by 3 
 * sum of 1,2,3,4
